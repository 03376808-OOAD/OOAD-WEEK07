Object คือ สิ่งที่เป็นรูปธรรมและนามธรรม ที่มีอยู่จริงบนโลกใบนี้ (Real world object)
	1.สิ่งที่เป็นรูปธรรม (tangible) คือ สิ่งที่สามารถจับต้องได้ สัมผัสได้ เช่น รถ สุนัข ต้นไม้ ใบรายการสินค้า เป็นต้น
	2.สิ่งที่เป็นนามธรรม (intangible) คือ สิ่งที่ไม่สามาถจับต้องได้ ไม่มีตัวตน เช่น ความเป็นเจ้าของ ความคิด ความรู้สึก เป็นต้น

ในด้านเทคโนโลยีคอมพิวเตอร์ คำว่า Object ถูกนำมาใช้ด้านการเขียนโปรแกรม ที่เรียกว่า โปรแกรมเชิงวัตถุ (OOP ย่อมาจาก Object Oriented Programming) 
ซึ่งหมายถึง การโปรแกรมที่เสมือนจริง มีลักษณะเสมือนธรรมชาติมากที่สุด แนวคิดของการเขียนโปรแกรมเชิงวัตถุนั้นให้ความสำคัญกับ ข้อมูล(data) และ พฤติกรรม(behavior) 
ของวัตถุและความสัมพันธ์กันระหว่างวัตถุ ต่างกับ แนวคิดการเขียนโปรแกรมแบบดังเดิมมักนิยมใช้ การเขียนโปรแกรมเชิงกระบวนการ (Procedural Programming) 
ซึ่งให้ความสำคัญกับขั้นตอนกระบวนการที่ทำ โดยแบ่งโปรแกรมออกเป็นส่วนๆตามลำดับขั้นตอนการทำงาน ในการเขียนโปรแกรมเชิงวัตถุ ในโปรแกรมหนึ่งๆมีหลาย Object 
โดยแต่ละ Object ก็จะมีความสัมพันธ์และเชื่อมโยงระหว่าง Object ด้วยกันเอง
ดังนั้นจึงจัด Object ให้เป็นกลุ่ม เรียกว่าClass

Class หมายถึง Object ที่มีโครงสร้างพื้นฐานพฤติกรรมเดียวกัน ดังนั้น Object ที่มีคุณสมบัติลักษณะเดียวกันนี้จะรวมกลุ่มอยู่ใน Class เดียวกัน
จึงสรุปได้ว่า Class คือ ต้นแบบข้อมูลที่มีไว้เพื่อสร้าง Object นั่นเอง Class นอกจากจะมีชื่อ Class ที่บอกคุณสมบัติของ Class แล้ว ยังมี Attribute และ Operation ต่างๆ ซึ่งเป็นตัวอธิบายรายละเอียด และหน้าที่ต่างๆด้วย
